# 📖 Project Glossary (Ubiquitous Language)

이 문서는 모든 프로젝트 모듈에서 공통으로 사용되는 **핵심 도메인 용어 사전**입니다.

---

## 1. Domain Entities
| 용어 (Korean) | 용어 (English) | 설명 |
| :--- | :--- | :--- |
| 당첨점/명당 | LottoSpot | 1등이 배출된 판매점 정보 |
| 내 번호 | MyNumbers | 사용자가 직접 저장하거나 생성한 번호 리스트 |
| 회차 | Round | 로또 추첨 회차 (e.g., 1125회) |

## 2. Technical Terms
- **WinningCheck**: 백엔드와 대조하여 당첨 여부를 판별하는 행위.
- **OCRScan**: 영수증 촬영을 통해 번호를 추출하는 행위.
