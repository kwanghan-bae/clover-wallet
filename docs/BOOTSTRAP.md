# 🍀 Clover Wallet Bootstrap Contract

이 환경은 **STATELESS**하며, 모든 컨텍스트는 문서로부터 재구성되어야 합니다.

## 2. Stateless 연속성 규칙
- 모든 진행 상황은 반드시 문서(`ROADMAP.md`, `WORK_LOG_*.md`)에 기록되어야 합니다.
- 불확실한 경로나 명세는 추측하지 말고 검색하거나 질문하십시오.
- **Stop -> Update Docs -> Ask**: 모르면 멈추고 문서를 본 뒤 물어보는 것이 원칙입니다.
