## 1. 지식 로딩 (Sequential Loading)
당신은 지금부터 다음 문서들을 **순서대로** 읽고, 각 문서를 읽을 때마다 **"핵심 제약 사항 1가지"**를 요약하여 출력하십시오. (대충 읽는 것을 방지합니다.)

1. `docs/backend/BOOTSTRAP.md`
2. `docs/backend/IDENTITY.md`
3. `docs/backend/CONSTRAINTS.md`
4. `docs/backend/ENVIRONMENT.md`
5. `docs/backend/GLOSSARY.md`
6. `docs/backend/sessions/next_session.md`

## 2. 사고 준비 (System 2 Initialization)
모든 문서를 읽었다면, 작업을 시작하기 전 `docs/backend/SCRATCHPAD.md`를 열어 현재 목표에 대한 당신의 설계 초안을 먼저 작성하십시오.

## 3. 선언 (The Pledge)
위 과정을 마친 뒤, 다음과 같이 답하고 작업을 시작하십시오:
> "인지 로딩 완료. backend의 자아를 수용했습니다. 규칙 위반 시 스스로 수정을 거부하겠습니다."
